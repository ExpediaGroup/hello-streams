# hello-streams :: ordercleaner

## Build

```
# from top level directory
$ cd ordercleaner
$ ./mvnw clean install
```

## Run
```
# from top level directory
$ cd ordercleaner
$ make run
```

- Open a browser to http://localhost:5050/graphiql


## Test
```
# from top level directory
$ cd ordercleaner
$ ./mvnw clean test
```
