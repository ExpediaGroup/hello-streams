# hello-streams :: orderprocessor

## Build

```
# from top level directory
$ cd orderprocessor
$ ./mvnw clean install
```

## Run
```
# from top level directory
$ cd orderprocessor
$ make run
```

- Open a browser to http://localhost:5000/graphiql


## Test
```
# from top level directory
$ cd orderprocessor
$ ./mvnw clean test
```
